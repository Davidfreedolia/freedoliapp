# FREEDOLIAPP — SEO Growth Strategy V1

## Status
Planned

## Goal

Define the SEO strategy to attract Amazon sellers searching for tools,
templates and operational solutions related to Amazon FBA operations.

The objective is to capture high-intent traffic and guide visitors
towards the FREEDOLIAPP trial.

---

# 1 — SEO PRINCIPLE

FREEDOLIAPP SEO will focus on **problem-driven search queries**.

Instead of generic blog content, the strategy targets searches where
Amazon sellers are actively looking for solutions.

Examples of queries:

- amazon profit calculator
- amazon fba spreadsheet
- supplier management amazon fba
- inventory tracker amazon seller
- purchase order template amazon

These searches indicate users who already have operational problems
that FREEDOLIAPP can solve.

---

# 2 — SEO CONTENT TYPES

Three main SEO page categories:

1. **Tools / Calculators**
2. **Templates**
3. **Guides**

---

# 3 — TOOLS / CALCULATORS

These pages provide interactive tools that solve common seller problems.

Examples:

- Amazon Profit Calculator
- Amazon FBA Margin Calculator
- Amazon Inventory Turnover Calculator

Each page should contain:

- a simple calculator
- explanation of the concept
- comparison with manual spreadsheets
- call to action to try FREEDOLIAPP

---

# 4 — TEMPLATES

Amazon sellers often search for operational templates.

Example pages:

- Amazon Purchase Order Template
- Supplier Management Spreadsheet
- Amazon Inventory Tracking Spreadsheet

Page structure:

- explanation of the template
- downloadable version
- explanation of limitations of spreadsheets
- call to action to manage operations in FREEDOLIAPP instead

---

# 5 — GUIDES

Guides explain operational workflows.

Examples:

- How to manage suppliers for Amazon FBA
- How to track Amazon profitability
- How to organize Amazon seller operations

These guides help position FREEDOLIAPP as an authority in Amazon operations management.

---

# 6 — CORE SEO LANDING PAGES

Main keyword pages describing the product.

Examples:

- amazon-operations-software
- amazon-sourcing-management
- amazon-seller-operations-platform

These pages directly explain the FREEDOLIAPP solution.

---

# 7 — FUNNEL CONNECTION

SEO pages should guide visitors toward the trial.

Example funnel:

```
SEO page
→ tool / template / guide
→ explanation of problem
→ CTA to start free trial
→ activation wizard
```

---

# 8 — INTEGRATION WITH LANDING ASSISTANT

Visitors arriving from SEO pages may have specific problems.

The Freedoli assistant can detect context and guide the visitor.

Example:

**User arrives from:** amazon-profit-calculator

**Assistant message:**

> Hi 👋 I see you're trying to calculate Amazon profitability.
> I can show you a better way to track it.

---

# 9 — LONG TERM OBJECTIVE

Build a library of SEO pages that attract Amazon sellers searching
for operational solutions.

Target future scale:

**10–30 high-quality pages** generating recurring organic traffic.

---

# 10 — SEO Page Structure

All SEO pages should follow the same structure.

1. **Title** (keyword focused)
2. **Short introduction** explaining the problem.
3. **Tool / template / guide content.**
4. **Explanation of the limitations of manual solutions**
   (spreadsheets, manual tracking, etc.).
5. **Introduction of FREEDOLIAPP** as the operational solution.
6. **Clear call to action.**

Example CTA:

> Start managing your Amazon operations in one place.
>
> **Start free trial.**
